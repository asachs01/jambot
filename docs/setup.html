Setup Guide

This guide will walk you through setting up JamBot for your Discord server. You'll need to create both a Discord bot and a Spotify application.

1Discord Bot Setup

Create a Discord Application

  1. Go to the Discord Developer Portal
  2. Click "New Application"
  3. Give it a name (e.g., "JamBot") and click "Create"

Configure Bot Settings

  1. In the left sidebar, click "Bot"
  2. Click "Add Bot" ‚Üí "Yes, do it!"
  3. Under "Privileged Gateway Intents", enable:
    • ‚úÖ Message Content Intent
    • ‚úÖ Server Members Intent
  4. Click "Save Changes"

Get Your Bot Token

  1. Under the "Token" section, click "Reset Token"
  2. Click "Copy" to copy your bot token
  3. Save this token - you'll need it for the .env file

Add Bot to Your Server

  1. In the left sidebar, click "OAuth2" ‚Üí "URL Generator"
  2. Under "Scopes", select:
    • ‚úÖ bot
    • ‚úÖ applications.commands
  3. Under "Bot Permissions", select:
    • ‚úÖ Read Messages/View Channels
    • ‚úÖ Send Messages
    • ‚úÖ Send Messages in Threads
    • ‚úÖ Read Message History
    • ‚úÖ Add Reactions
    • ‚úÖ Use Slash Commands
  4. Copy the generated URL at the bottom
  5. Paste the URL in your browser and select your Discord server
  6. Click "Authorize"
‚úÖ Discord Setup Complete! Your bot should now appear in your server (offline until you run it).

2Spotify API Setup

Create a Spotify Application

  1. Go to the Spotify Developer Dashboard
  2. Log in with your Spotify account
  3. Click "Create app"
  4. Fill in the form:
    • App name: JamBot
    • App description: Discord bot for bluegrass jam playlists
    • Redirect URI: http://localhost:8888/callback
    • API/SDKs: Select "Web API"
  5. Agree to the terms and click "Save"

Get Your Spotify Credentials

  1. Click on your newly created app
  2. Click "Settings"
  3. You'll see:
    • Client ID - Copy this
    • Client Secret - Click "View client secret" and copy it
  4. Save both of these - you'll need them for authentication

Get Your Refresh Token

JamBot includes a helper script to get your Spotify refresh token:

# 1. Create your .env file
cp .env.example .env

# 2. Add your Discord and Spotify credentials to .env
# DISCORD_BOT_TOKEN=your_discord_token
# SPOTIFY_CLIENT_ID=your_client_id
# SPOTIFY_CLIENT_SECRET=your_client_secret
# SPOTIFY_REDIRECT_URI=http://localhost:8888/callback

# 3. Run the Spotify auth helper
python spotify_auth.py

# 4. Follow the prompts:
# - A browser window will open
# - Log in to Spotify and authorize the app
# - You'll be redirected (the page will show an error - that's normal!)
# - Copy the refresh token from the terminal output

# 5. Add the refresh token to your .env file
# SPOTIFY_REFRESH_TOKEN=your_refresh_token_here

3Configuration

Your .env file should now look like this:

# Discord Configuration
DISCORD_BOT_TOKEN=your_discord_bot_token_here

# Spotify Configuration
SPOTIFY_CLIENT_ID=your_spotify_client_id_here
SPOTIFY_CLIENT_SECRET=your_spotify_client_secret_here
SPOTIFY_REDIRECT_URI=http://localhost:8888/callback
SPOTIFY_REFRESH_TOKEN=your_spotify_refresh_token_here

# Database Configuration (optional)
DATABASE_PATH=/app/data/jambot.db

# Logging Configuration (optional)
LOG_LEVEL=INFO
LOG_FILE=/app/logs/jambot.log

4Run JamBot

Using Python Directly

# Install dependencies
pip install -r requirements.txt

# Run the bot
python -m src.main

Using Docker

# Build and run with Docker Compose
docker-compose up -d

# View logs
docker-compose logs -f jambot
